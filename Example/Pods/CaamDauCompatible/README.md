# Compatible
> 蚕豆 (粤语caam dau)

> 前言：CaamDau是一个开发套件，旨在构建一个通用业务组件库和Cocoa便利性扩展，作为基础业务组件，在新项目初期与老项目维护期快速完成通用业务代码的构建（其实就是方便自己，同时也希望能方便同志）。

CaamDauCompatible 提供一个 CaamDau 的域
```
pod 'CaamDauCompatible'

pod 'CaamDau/Compatible'
```
